# BasedNode Installation Guide (Unofficial BF1337 Fork)

Welcome to the **BasedNode Unofficial Install Guide**!

This guide helps you install and run a BasedNode on **Ubuntu under WSL (Windows Subsystem for Linux)**.  
Perfect for Windows users new to Linux.

Typical install time: **~42 minutes**.

---

## 🚀 Why WSL?

- Run Ubuntu directly inside Windows
- No virtual machines needed
- Great for local tests and learning

⚠️ **Note:**  
WSL isn’t ideal for running a node 24/7 or in production. For that, use a VPS or dedicated server.

---

## ✅ How to install Ubuntu (WSL)

1. Open the **Microsoft Store** and search for **Ubuntu 22.04 LTS**.
2. Click **Install**.
3. Launch Ubuntu from the Start menu.
4. Choose a Linux username and password.

---

## ✅ Who is this guide for?

Use this guide if:

- You want to try running a BasedNode
- You’re testing or learning
- You’re not running a public node yet

---

## ⚠️ Security Reminder

Running a node for testing is safe.  
But avoid running a node 24/7 on the same machine that stores significant crypto funds.

---

## ✅ How to run the install script

Once Ubuntu is installed and running, refer to the `basednode_install.sh` script for all installation steps.
